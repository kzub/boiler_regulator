This is a sophisticated DIY project involving motion control, networking, user interface, and sensor integration.

Below is a complete guide covering compatibility, PlatformIO setup, code structure, and wiring.

1. Device Compatibility Check (Critical)

There is a Major Hardware Conflict in your component list that requires a specific workaround.

●      The Conflict: You are using an Arduino Leonardo. On the Leonardo, the I2C pins (SDA, SCL) are located on Digital Pins 2 and 3.

●      ** The Problem:** The CNC Shield R3 is designed for the Arduino Uno. It hardwires the X-Axis and Y-Axis stepper drivers to Pins 2, 3, 5, and 6.

○      If you use the X and Y slots for your motors, you cannot use the I2C LCD because the pins are occupied by the motors.

●      The Solution: You must use the Z-Axis and A-Axis slots on the CNC Shield for your two motors.

○      Z-Axis uses Pins 4 and 7.

○      A-Axis uses Pins 12 and 13.

○      This leaves Pins 2 and 3 free for your LCD Display.

Correction on Sensors: You mentioned "three I2C temperature sensors (DS18B20)". The DS18B20 is NOT an I2C sensor; it uses the OneWire protocol. This is fine, but you cannot connect them to the I2C bus. They will need a separate digital pin (e.g., A0) and a 4.7kΩ pull-up resistor.

3. Logic Modules and Code

We will split the code into logical parts. Create a src/main.cpp file.

Key Logic Flow:

1.     Setup: Init Ethernet, LCD, Sensors.

2.     Homing: Motors move towards endstops. Count steps. Once hit, move back the same amount.

3.     Loop:

○      Check HTTP Client (Ethernet).

○      Read Encoder/Buttons.

○      Update LCD (based on menu state).

○      Run Steppers (must be called frequently).

○      Read Temperatures (periodically, not every loop).

The Code (src/main.cpp)

 

C++

 

 

#include <Arduino.h>
#include <SPI.h>
#include <Ethernet.h>
#include <Wire.h>
#include <LiquidCrystal_I2C.h>
#include <OneWire.h>
#include <DallasTemperature.h>
#include <AccelStepper.h>

// --- CONFIGURATION ---
// MAC Address (sticker on your board or make one up)
byte mac[] = { 0xDE, 0xAD, 0xBE, 0xEF, 0xFE, 0xED };
IPAddress ip(192, 168, 1, 177); // Manual IP

// Pins (Mapped for Leonardo + CNC Shield Z/A Axis workaround)
#define ONE_WIRE_BUS A0   // DS18B20 Data pin
#define BUTTON_PIN   A1   // Menu Button
#define ENCODER_A    A2
#define ENCODER_B    A3

// Stepper 1 (Valve 1) - Using Z Axis Driver Slot
#define M1_STEP_PIN  4
#define M1_DIR_PIN   7
#define M1_ENDSTOP   11   // Z- limit pin on shield

// Stepper 2 (Valve 2) - Using A Axis Driver Slot
#define M2_STEP_PIN  12
#define M2_DIR_PIN   13
#define M2_ENDSTOP   9    // SpnEn (often used for A) or reuse X/Y limits (9/10). *Check wiring below*

// Settings
#define MAX_POSITION 2000 // Max steps allowed away from endstop
#define HOMING_SPEED 200.0
#define RUN_SPEED    400.0
#define ACCEL        200.0

// --- OBJECTS ---
EthernetServer server(80);
LiquidCrystal_I2C lcd(0x27, 16, 2);
OneWire oneWire(ONE_WIRE_BUS);
DallasTemperature sensors(&oneWire);

// 1 = Driver Interface
AccelStepper stepper1(AccelStepper::DRIVER, M1_STEP_PIN, M1_DIR_PIN);
AccelStepper stepper2(AccelStepper::DRIVER, M2_STEP_PIN, M2_DIR_PIN);

// Global Variables
enum MenuState { STATE_DEFAULT, STATE_EDIT_M1, STATE_EDIT_M2 };
MenuState currentState = STATE_DEFAULT;

float temp1 = 0, temp2 = 0, temp3 = 0;
long lastTempTime = 0;
int lastEncoderVal = 0; // Simple polling for encoder

// Button Debounce
int lastButtonState = HIGH;
unsigned long lastDebounceTime = 0;

void homingRoutine(AccelStepper &stepper, int endstopPin) {
  // Move until endstop is hit
  stepper.setMaxSpeed(HOMING_SPEED);
  stepper.setSpeed(-HOMING_SPEED); // Move towards minimal

  long stepsCounted = 0;
 
  // Keep running until switch is triggered (assuming LOW when pressed)
  // Note: Add a safety timeout in production code!
  while (digitalRead(endstopPin) == HIGH) {
    stepper.runSpeed();
    stepsCounted++;
    delayMicroseconds(100); // Small delay to allow mechanics to move
  }

  stepper.setCurrentPosition(0); // This is now position 0
 
  // "Return back counted steps to restore actual position"
  // NOTE: This logic assumes the valve was open X steps before power loss.
  // Since we don't know where it was before power loss, we can only
  // move it to a safe default OR stay at 0.
  // If you meant "Measure how far we moved to hit the wall, then go back":
 
  stepper.moveTo(stepsCounted);
  while(stepper.distanceToGo() != 0) {
    stepper.run();
  }
}

void setup() {
  // 1. Init Serial
  Serial.begin(115200);

  // 2. Init Pins
  pinMode(M1_ENDSTOP, INPUT_PULLUP);
  pinMode(M2_ENDSTOP, INPUT_PULLUP);
  pinMode(BUTTON_PIN, INPUT_PULLUP);
  pinMode(ENCODER_A, INPUT_PULLUP);
  pinMode(ENCODER_B, INPUT_PULLUP);

  // 3. Init LCD
  lcd.init();
  lcd.backlight();
  lcd.setCursor(0,0);
  lcd.print("Init System...");

  // 4. Init Sensors
  sensors.begin();

  // 5. Init Steppers
  stepper1.setMaxSpeed(RUN_SPEED);
  stepper1.setAcceleration(ACCEL);
  stepper2.setMaxSpeed(RUN_SPEED);
  stepper2.setAcceleration(ACCEL);

  // 6. Homing
  lcd.setCursor(0,1);
  lcd.print("Homing M1...");
  homingRoutine(stepper1, M1_ENDSTOP);
 
  lcd.setCursor(0,1);
  lcd.print("Homing M2...");
  homingRoutine(stepper2, M2_ENDSTOP);

  // 7. Init Ethernet
  Ethernet.begin(mac, ip);
  server.begin();
 
  lcd.clear();
}

void handleWeb() {
  EthernetClient client = server.available();
  if (client) {
    boolean currentLineIsBlank = true;
    String header = "";
    while (client.connected()) {
      if (client.available()) {
        char c = client.read();
        if (header.length() < 100) header += c;

        if (c == '\n' && currentLineIsBlank) {
          // Parse request
          // Example: GET /?m1=1000 HTTP/1.1
          int idxM1 = header.indexOf("m1=");
          int idxM2 = header.indexOf("m2=");
         
          if(idxM1 > 0) {
            int val = header.substring(idxM1+3).toInt();
            if(val >= 0 && val <= MAX_POSITION) stepper1.moveTo(val);
          }
          if(idxM2 > 0) {
             int val = header.substring(idxM2+3).toInt();
             if(val >= 0 && val <= MAX_POSITION) stepper2.moveTo(val);
          }

          // Send Response
          client.println("HTTP/1.1 200 OK");
          client.println("Content-Type: text/plain");
          client.println("Connection: close");
          client.println();
          client.print("T1:"); client.print(temp1);
          client.print("|T2:"); client.print(temp2);
          client.print("|T3:"); client.print(temp3);
          client.print("|M1:"); client.print(stepper1.currentPosition());
          client.print("|M2:"); client.print(stepper2.currentPosition());
          break;
        }
        if (c == '\n') currentLineIsBlank = true;
        else if (c != '\r') currentLineIsBlank = false;
      }
    }
    delay(1);
    client.stop();
  }
}

void readEncoder() {
  // Basic rotary encoder logic
  static int lastA = HIGH;
  int currentA = digitalRead(ENCODER_A);
 
  if (lastA == HIGH && currentA == LOW) {
    int change = 0;
    if (digitalRead(ENCODER_B) == LOW) change = 50; // Clockwise
    else change = -50; // Counter-Clockwise
   
    if (currentState == STATE_EDIT_M1) {
      long newPos = stepper1.targetPosition() + change;
      if(newPos >= 0 && newPos <= MAX_POSITION) stepper1.moveTo(newPos);
    }
    else if (currentState == STATE_EDIT_M2) {
      long newPos = stepper2.targetPosition() + change;
      if(newPos >= 0 && newPos <= MAX_POSITION) stepper2.moveTo(newPos);
    }
  }
  lastA = currentA;
}

void handleMenu() {
  // Read Button
  int reading = digitalRead(BUTTON_PIN);
  if (reading == LOW && lastButtonState == HIGH && millis() - lastDebounceTime > 50) {
     // Toggle State
     if(currentState == STATE_DEFAULT) currentState = STATE_EDIT_M1;
     else if(currentState == STATE_EDIT_M1) currentState = STATE_EDIT_M2;
     else currentState = STATE_DEFAULT;
     lcd.clear();
     lastDebounceTime = millis();
  }
  lastButtonState = reading;

  // Display Logic
  static long lastLCDUpdate = 0;
  if(millis() - lastLCDUpdate > 200) {
    if(currentState == STATE_DEFAULT) {
      lcd.setCursor(0,0);
      lcd.print(temp1, 1); lcd.print(" P1:"); lcd.print(stepper1.currentPosition());
      lcd.setCursor(0,1);
      lcd.print(temp3, 1); lcd.print(" P2:"); lcd.print(temp2, 1); // Layout as requested
    } else if (currentState == STATE_EDIT_M1) {
       lcd.setCursor(0,0); lcd.print(">> EDIT MOTOR 1");
       lcd.setCursor(0,1); lcd.print("Pos: "); lcd.print(stepper1.targetPosition());
    } else {
       lcd.setCursor(0,0); lcd.print(">> EDIT MOTOR 2");
       lcd.setCursor(0,1); lcd.print("Pos: "); lcd.print(stepper2.targetPosition());
    }
    lastLCDUpdate = millis();
  }
}

void loop() {
  // 1. Critical: Run Steppers (must happen as fast as possible)
  stepper1.run();
  stepper2.run();

  // 2. Read Temps (Every 2 seconds)
  if(millis() - lastTempTime > 2000) {
    sensors.requestTemperatures();
    temp1 = sensors.getTempCByIndex(0);
    temp2 = sensors.getTempCByIndex(1);
    temp3 = sensors.getTempCByIndex(2);
    lastTempTime = millis();
  }

  // 3. Handle Inputs
  handleWeb();
  readEncoder();
  handleMenu();
}

4. Wiring Connection Guide

This is the most complex part due to the Leonardo/CNC Shield conflict. You must ignore the labels "X" and "Y" on the shield.

A. CNC Shield & Motors

Insert the A4988 drivers into the Z and A slots. (Ensure the potentiometer on the driver faces away from the axis pins, check your shield version).

Component

CNC Shield/Arduino Pin

Description

Motor 1 (Stepper)

Z-Axis Header

Connect 4 wires of Motor 1 here

Motor 2 (Stepper)

A-Axis Header

Connect 4 wires of Motor 2 here

Motor 1 Endstop

Z- (Pin 11)

Connect Switch between Z- (White) and GND (Black)

Motor 2 Endstop

SpnEn (Pin 12) or D9

The A-axis pins are tricky on generic shields. Recommendation: Connect the endstop directly to Arduino Pin 9 and GND (solder to the header pins sticking out of the shield). Update code: #define M2_ENDSTOP 9

External Power

Blue Terminal Block

Connect 12V-24V Power Supply (Required for motors)

B. Sensors (DS18B20)

Do not plug these into I2C.

1.     Connect all VCC pins to +5V.

2.     Connect all GND pins to GND.

3.     Connect all Data pins together to Arduino Analog Pin A0.

4.     Crucial: Place a 4.7kΩ resistor between the Data line and VCC (+5V).

C. LCD (I2C)

1.     SDA: Connect to Arduino Pin 2.

2.     SCL: Connect to Arduino Pin 3.

3.     VCC: +5V.

4.     GND: GND.
(Note: Because we used Z/A slots for motors, pins 2 and 3 are free for this!)

D. Controls (Encoder & Button)

1.     Menu Button: One side to A1, other side to GND.

2.     Encoder A: To A2.

3.     Encoder B: To A3.

4.     Encoder GND: To GND.
